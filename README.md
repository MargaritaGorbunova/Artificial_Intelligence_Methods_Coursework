# Artificial Intelligence Methods Coursework

AI Methods Course Labs (MUCTR). Python, PyTorch, TensorFlow implementation of neural networks: perceptrons, CNN/MLP comparison, image classification, regression tasks.

## Laboratory Works

### Lab 01: Concrete Strength Prediction
**Topic:** Implementation of single-layer and multi-layer perceptrons from scratch  
**Task:** Regression  
**Dataset:** [Concrete Compressive Strength](https://www.kaggle.com/datasets/elikplim/concrete-compressive-strength-data-set) on Kaggle  
**Key Points:** Widrow-Hoff algorithm, backpropagation, early stopping  
**Result:** Multi-layer perceptron reduced MSE by 15%

### Lab 02: Air Quality Classification
**Topic:** Multiclass classification with imbalanced data handling  
**Task:** Classification (4 classes)  
**Dataset:** [Air Quality and Pollution Assessment](https://www.kaggle.com/datasets/mujtabamatin/air-quality-and-pollution-assessment) on Kaggle  
**Key Points:** PyTorch, GPU training, confusion matrix analysis  
**Framework:** PyTorch

### Lab 03: CNN vs MLP Comparison on Fashion MNIST
**Topic:** Computer Vision, architecture comparison  
**Task:** Image classification  
**Dataset:** Built-in Fashion MNIST dataset in TensorFlow/Keras  
**Key Points:** TensorFlow, CNN vs MLP, data augmentation  
**Result:** CNN (89.7%) outperformed MLP (86.5%)

### Lab 04: Butterfly Image Classification
**Topic:** Working with real-world data, augmentation techniques  
**Task:** Image classification  
**Dataset:** [Butterfly Image Classification](https://www.kaggle.com/datasets/phucthaiv02/butterfly-image-classification) on Kaggle  
**Key Points:** Keras, ImageDataGenerator, Kaggle API  
**Result:** 73% accuracy on test set

### Lab 05: MNIST Digit Classification
**Topic:** Classical Computer Vision problem  
**Task:** Digit classification  
**Dataset:** Built-in MNIST dataset in TensorFlow/Keras  
**Key Points:** CNN, data augmentation  
**Result:** 97.45% accuracy
